A generator for the AST from BNF grammar files

An abstract base class is created that contains the specific subclasses with the appropriate arguments.

The baseclass also contains a Visitor-Interface which has call functions for every Expression type. These also each implement the abstract Expression.accept function calling their specific call function. The visitor pattern is used to be able to use functions on the specific Expression types without changing the generated class.
